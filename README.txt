ShelfLister 3.0 from the University of Texas at Arlington
  modified by the University of Pittsburgh, University Library System.

See the original source at:
	http://rocky.uta.edu/doran/shelflister/

Modified by the Pitt ULS to enable the existing checkboxes to allow for batch
item marking, optionally identified by user (if REMOTE_USER is set).

Additional header display of the marked items allows for a count and clearing
of the current datafile.
